ENSEEIHT • 2025-2026 Analog Systems Project

FM Optoelectronic Link: Emitter & Receiver Design

Design and hardware implementation of a complete frequency-modulated (FM) optical communication system for audio transmission. This project involved the precise sizing of modulation, demodulation, and power stages.

Emitter: FM Modulation & LED Driving

  • Input Conditioning: Active audio filter boosting high frequencies to improve signal-to-noise ratio (SNR) before modulation.
  • FM Modulation: Voltage-Controlled Oscillator (VCO) centered at carrier frequency f₀, converting audio signals into frequency deviations.
  • Optical Transmission: LED driver circuit (Bipolar/MOS) optimizing linearity and rise time for reliable IR transmission.

Receiver: Demodulation & Power Output

  • Transimpedance: Photodiode current conversion followed by high-gain cascode and differential pair amplification with signal clipping.
  • FM Demodulation: Phase-Locked Loop (PLL) system for accurate frequency-to-voltage conversion.
  • Signal Filtering: Active filter removing high-frequency harmonics to recover the original audio spectrum.
  • Power Stage: Class AB Push-Pull amplifier providing high current gain to drive the speaker load.

ENSEEIHT • 2025-2026 Embedded Systems Project

RFID-Based Access Control System

Development of an automated security system using a PIC18F microcontroller. The project integrates an RFID reader and a servo-controlled door mechanism, focusing on real-time embedded communication and hardware-software synchronization.

Embedded Hardware & Communication

  • RFID Interfacing: Establishing robust communication between the PIC18F and a CR95HF module using the UART protocol.
  • Protocol Management: Implementing NFC tag detection and identification by parsing hex-command responses from the RFID click.
  • Actuator Driving: Implementing a PWM (Pulse Width Modulation) signal to control precise servo-motor angles for door operation.

Firmware & Control Logic

  • Low-Level C Development: Programming the control loop in MPLAB X, optimizing register-level configurations for ADC and Timers.
  • Interrupt Handling: Utilizing hardware interrupts for efficient tag polling and event management without blocking the main CPU.
  • Safety Features: Implementation of identity validation routines and integration of a temperature-based security interlock.

ENSEEIHT • 2025-2026 Digital Systems Project

FPGA Digital Front-End & Frequency Meter

Design and VHDL implementation of a high-speed digital signal processing chain on an Artix-7 FPGA. The system acquires optical "sawtooth" fringe signals to calculate and display their frequency in real-time.

Digital Signal Processing (DSP)

  • Fixed-Point Filtering: Implementation of a 1st-order High-Pass filter (10kHz cutoff) using Euler approximation and optimized bit-truncation.
  • Edge Enhancement: Hardware-based derivation and dynamic thresholding to convert analog fringes into a robust binary pulse signal.
  • Data Acquisition: Interfacing a native 12-bit XADC (1MHz sampling) and an external 12-bit DAC for real-time signal monitoring.

Logic Design & Architecture

  • VHDL Modeling: Structural and behavioral coding of mathematical operators, registers, and synchronous state machines.
  • Frequency Measurement: Implementation of a high-resolution frequency counter with a 100ms refresh rate and 4-digit precision.
  • Hardware Interfacing: Management of 7-segment display multiplexing and Pmod DA2 communication protocols.

ENSEEIHT • 2024-2025 IoT Project

IoT Heart Monitor Development

Implementation of a complete photoplethysmography (PPG) acquisition chain. This project required designing a custom analog front-end, digitizing signals via FPGA, and performing automated diagnostics.

Analog & Mixed-Signal Design

  • 3-Stage Amplifier Sizing: Engineered a custom chain featuring a Common Drain stage for high input impedance, a Differential Pair for precise gain, and a Common Collector for low output impedance. Designed the whole PCB on PSpice.
  • Analog Filtering: Implementation of a Rauch active filter to eliminate 50Hz electrical noise and define the frequency bandwidth for PPG signals.
  • ADC Design: Development of an 8-bit Single-Slope ADC implemented on a DE1-SoC FPGA via Quartus.

Digital Signal Processing & Analysis

  • Matlab Processing: Signal filtering using Savitzky-Golay (FIR) and Butterworth (IIR) algorithms for optimal noise reduction.
  • Pulse Wave Velocity: Extraction of physiological parameters, including systolic peak detection and heart rate calculation.
  • Statistical Diagnosis: Implementation of Neyman-Pearson tests for automated detection of tachycardia, bradycardia, and arrhythmia.
